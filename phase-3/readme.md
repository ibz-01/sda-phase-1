
=== GENERIC CONCURRENT REAL-TIME PIPELINE ===

MAIN FILE: main.py
Run with: python main.py

DIRECTORY SETUP:
- Place your CSV data file inside the  data/  folder
- Update "dataset_path" in config.json to point to your file
- Update "schema_mapping" columns in config.json to match your CSV headers

DEPENDENCIES:
Install with:  pip install matplotlib

PYTHON VERSION: 3.8+

CONFIG FILE: config.json (must be in same folder as main.py)
